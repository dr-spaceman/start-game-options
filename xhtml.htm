Linking to another page
Some site <a href="http://some-site.com/">Some site</a>
Text Formatting
Bold Text <b>Bold Text</b>
Italic Text <i>Italic Text</i>
Strikethrough <strike>Strikethrough</strike>
Frickin Huge Text <big>Frickin Huge Text</big>
Teeny Tiny Text <small>Teeny Tiny Text</small>
Special Formatting
Use BLOCKQUOTE for quoted words, ie: <blockquote>Blah blah blah</blockquote>
Hide spoilers with DEL, ie: <del>Somebody totally fucking died</del>
Images
<img src="http://blah.com/image.jpg" alt="An image"/>
Note: Make sure you surround the image URL with quotes and include an ALT tag also surrounded with quotes (the double-quote character (") must be used).
Don't forget the forward slash (/) at the end of the tag!
For more on XHTML, see W3Schools' XHTML Tutorial.
